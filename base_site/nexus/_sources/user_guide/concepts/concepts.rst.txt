Concepts
===============

What is Quantinuum Nexus?
-------------------------

Quantinuum Nexus is a cloud-based platform that enables users to seamlessly run,
review, and collaborate on quantum computing projects.

The platform integrates support for various quantum hardware providers using the
pytket quantum programming tools to optimize circuit performance and translation
between different backends.
