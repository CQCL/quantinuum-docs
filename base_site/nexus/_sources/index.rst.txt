Quantinuum Nexus Documentation
=============================================

This documentation provides an overview of the `Quantinuum Nexus platform <https://nexus.quantinuum.com>`_.

- **Nexus Concepts**: explains the main concepts behind Nexus, such as projects and jobs.
- **Guides**: covers basic Nexus usage for users and administrators, mostly focusing on the web user interface.
- **Trainings**: provides examples and tutorials on how to use Nexus capabilities, mostly via the Python client, ``qnexus``.
- **API Reference**: lists the Python API documentation for the ``qnexus`` package.
